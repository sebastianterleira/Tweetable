# INSTALLATION

Use the package manager `bundle` to install foobar

```bash
$ bundle install
```

```bash
$ rails db:setup
```

Use `rails db:setup` doesn't run the migrations so you'll have to run them manually

```bash
$ rails db:migrate
```
